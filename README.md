# 🚀 ProofOfWork

ProofOfWork is a full-stack developer progress tracker that helps developers document, showcase, and track their daily learning journey. Instead of relying solely on GitHub contributions, users can post their work with proof links, time invested, and descriptions to build a meaningful portfolio of their progress.

## ✨ Features

- 🔐 JWT Authentication
- 👤 User Registration & Login
- 📝 Post Daily Proof of Work
- 🔗 Attach GitHub, Project, or Article Links
- ⏱️ Track Time Spent Coding
- 📊 Dashboard with Weekly Activity
- 🔥 Coding Streak Tracker
- 📂 Public Developer Portfolio
- 📈 Progress Analytics

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/<your-username>/ProofOfWork.git
cd ProofOfWork
```

### Backend

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

pip install -r requirements.txt

uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npx expo start
```
